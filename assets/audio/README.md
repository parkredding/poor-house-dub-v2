# Audio Assets

This directory contains audio files for the Dub Siren.

## Custom Secret Mode MP3

Place your custom MP3 file here with the filename:
```
custom.mp3
```

This file will be played when you trigger the Custom Audio secret mode by rapidly pressing the shift button 4 times.

**Requirements:**
- Format: MP3
- Recommended: Mono or stereo
- Sample rate: Any (will be resampled to 44.1kHz)

**Note:** If the file is not found, the secret mode will activate but no audio will play.
